# VelocityMall 极速商城

基于 Spring Cloud Alibaba 的秒杀电商微服务项目，10 模块，Java 17 + Spring Boot 3.2。

## 模块

| 模块 | 端口 | 职责 |
|------|------|------|
| gateway | 8080 | 统一入口、JWT 鉴权、Sentinel 限流 |
| product | 8081 | 商品/分类/库存 |
| order | 8082 | 订单/购物车/支付 |
| seckill | 8083 | Redis Lua 秒杀 |
| search | 8085 | Elasticsearch 搜索 |
| coupon | 8086 | 优惠券 |
| review | 8087 | 评价/互动 |
| user | 8088 | 用户/登录 |
| admin | 8089 | 后台管理 |

## 秒杀压测结果

### 基础能力验证

| 能力 | VU | QPS | 结果 |
|------|:---:|------|------|
| Sentinel 限流 | 1,000 | 1,992 | 99.75% 请求 429 拦截 |
| 防超卖 | 200 | 1,561 | 50 库存精确售出，零超卖，P95=4ms |
| 熔断降级 | 500 | 1,146 | 1s 内熔断，失败率 0.01% |

### 集群极限压测

| 轮次 | VU | 吞吐量 | P95 | 秒杀成功 | 超卖 |
|------|:---:|------|------|:---:|:---:|
| 300 VU | 300 | **1,897 req/s** | 185ms | 1,000 | 0 |
| 1500 VU | 1,500 | **2,142 req/s** | 1.22s | 1,000 | 0 |

架构成 Gateway×3 + Seckill×3 混合部署，Sentinel 全解封，瓶颈在 Windows TCP 连接上限。
